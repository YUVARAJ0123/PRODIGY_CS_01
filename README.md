Caesar Cipher Implementation

Project Overview

This project implements the Caesar Cipher algorithm using Python. The Caesar Cipher is a classical encryption technique that shifts each letter in a message by a fixed number of positions in the alphabet.

Objective

To develop a Python application that encrypts and decrypts messages using a user-defined shift value while demonstrating the fundamentals of cryptography.

Features

- Encrypt text messages
- Decrypt encrypted messages
- User-defined shift value
- Supports uppercase and lowercase letters
- Simple and efficient implementation

Technologies Used

- Python 3
- Visual Studio Code (VS Code)
- GitHub

Sample Output

Input Message: HELLO

Shift Value: 3

Encrypted Message: KHOOR

Decrypted Message: HELLO
